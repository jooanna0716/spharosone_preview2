# MY CLOUD MEMBERSHIP Website

## 1. Project Description
쿠폰 관리 및 멤버십 관리를 위한 어드민 대시보드 웹사이트. 관리자를 위한 쿠폰 발급/조회/관리와 멤버십 회원 관리 기능을 제공하는 B2B SaaS 스타일의 공식 소개 사이트.

## 2. Page Structure
- `/` - Home (서비스 소개, 주요 기능 하이라이트)
- `/about` - About (서비스 소개, 팀/브랜드 소개)
- `/contact` - Contact (문의 폼, 연락처)

## 3. Core Features
- [x] 상단 네비게이션 바 (로고, 메뉴, 로그인 버튼)
- [x] Home 페이지 (Hero + 주요 기능 소개)
- [x] About 페이지 (서비스/팀 소개)
- [x] Contact 페이지 (문의 양식)
- [x] Footer (링크, 저작권 표시)

## 4. Data Model Design
Supabase 미연결 상태 - 목업 데이터 활용

- Stripe: 미연결
